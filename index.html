Synchronized Audio Manipulation

Synchronized Audio Manipulation is a web application designed for synchronizing audio output of clients and enabling the manipulation of sound behavior using focal point analysis. In the current implementation, songs are gathered from a user selected library on their local machine. The back-end development was done in Java, and the front-end development was done using Javascript. The WebRTC API was used for sound synchronization and the SocketIO API was used for server communication.

This project was developed as the final project for the Introduction to Software Engineering course (CS 32). It was a group project that involved Elias Ellison, Young Rae-Kim and Vanya Cohen. The project was completed in the Spring of 2015, but the group continually attempts to maintain and improve the application.

  • Source: Github
  • Date: May 2015

Graph Manager

Graph Manager is an Android Application that allows for data visualization of InfluxDB timeseries. The application retrieves InfluxDB data using an encoded HTTP Get call and the data is returned in JSON form. "Android Graphview" is used to model the dataset, allowing the data to be scrollable and scalable.

The graphs are managed by a two pages: a list of graphs, each of which are contained themselves by a "graphlist list." This feature allows for easy navigation and organization.

  • Source: Github
  • Date: August 2014

Maps

Maps is a web application that visualizes a dataset of SQL latitude/longitude points and provides many of the same features as Google Maps. This includes shortest distance calculation, traffic modeling, and more. The backend of this web application was developed in Java, and the frontend was done using Javascript (in conjunction with HTML, CSS and jQuery). This was a partner project, completed with Andy Thompson (https://github.com/sfandy) for Brown's Introduction to Software Engineering course (CS 32).

At the culmination of the project, we succesfully built a web application that was able to efficiently query an SQL database for map coordinates and roads, project these 3-dimensional points onto a 2-dimensional canvas, autocorrect user input, and add traffic data to the existing map.

As per Brown's Department of Computer Science Collaboration Policy, it is impermissible to make this project publicly available.

  • Date: April 2015
